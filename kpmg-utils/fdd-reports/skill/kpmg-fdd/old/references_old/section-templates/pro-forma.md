# Pro forma and run-rate adjustments template

## Purpose
Document run-rate changes and pro forma adjustments with clear labeling and separation from historical facts.

## Recommended structure

### What is pro forma vs historical
- Clearly separate: historical results (fact) vs pro forma (scenario/assumption)

### Pro forma adjustments list
For each:
- Description
- Basis (contract, board-approved plan, management estimate)
- Timing (already implemented vs planned)
- Expected impact (range if needed)

### Sensitivities
- Key variables that could change the outcome
- Dependencies and execution risks

### Open items
- Evidence needed to validate run-rate actions

## Common pitfalls
- Presenting pro forma numbers as historical or “certain”.
