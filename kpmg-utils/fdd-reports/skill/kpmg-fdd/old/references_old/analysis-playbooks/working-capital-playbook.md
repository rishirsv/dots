# Working capital playbook

## Goal
Assess normalized working capital behavior and implications for cash flow and peg.

This playbook aligns with the diligence checklist workstream “Working capital analysis — normalized vs. actual”.

## Inputs to request
- Monthly/quarterly balance sheet detail for WC accounts
- AR/AP aging reports
- Inventory detail (if relevant)
- Deferred revenue detail (if applicable)
- Definition of working capital used in the deal (if provided)

## Core analyses
1) **Definition and scope**
- Specify included/excluded accounts and why.

2) **Trend and seasonality**
- Prefer monthly/quarterly view; note limitations if only period-end snapshots exist.

3) **Normalization**
- Provide a normalized estimate or range with basis.
- If not possible, clearly state what prevents normalization.

## Tie-outs and checks
- Ensure the WC definition used in narrative matches the exhibit.
- If using cash flow linkage, ensure signs and units are consistent.

## Output artifacts
- WC definition table
- Normalization exhibit (trend + normalized vs actual)
- Peg sensitivity notes (qualitative unless data supports quantification)
