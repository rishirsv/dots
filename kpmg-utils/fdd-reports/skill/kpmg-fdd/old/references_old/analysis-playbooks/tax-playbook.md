# Tax playbook

## Goal
Summarize tax structure, exposures, and key diligence questions.

This playbook aligns with the diligence checklist workstream “Tax structure and exposure”.

## Inputs to request
- Tax returns (recent years)
- Audit/notice correspondence
- Entity structure and jurisdictions
- NOL schedules (if applicable)

## Core analyses
- Identify key open questions and exposures.
- Separate confirmed facts from areas requiring specialist review.

## Output artifacts
- Tax exposures table (issue, status, potential magnitude, mitigant, basis)
- Open items list
