# Release Notes Guidelines

Rules and examples for writing engaging App Store release notes (What's New).

## Structure

Three optional sections. Only include sections with content and omit empty ones.

- `New`: new features or capabilities
- `Improved`: enhancements to existing features
- `Fixed`: bug fixes users would notice

## The 170-character rule

The first ~170 characters of What's New are visible on the app page without tapping "more." This is the hook.

- Lead with the single most impactful change.
- Write a complete, compelling sentence rather than a chopped list.
- Assume many users will never tap "more."

Example hook:

`Search just got faster. Find what you need in seconds, with smoother navigation and smarter reminders throughout the app.`

## Tone

- Be benefit-focused, not implementation-focused.
- Be engaging without fluff.
- Use direct address when it helps.
- Prefer action verbs and concrete improvements.
- Avoid vague filler like "various improvements."

## Anti-patterns

- `Bug fixes and improvements`
- headings with version numbers
- competitor mentions
- references to other platforms
- keyword stuffing
- empty marketing hype
- giant walls of text

## Keyword echo

What's New is not indexed for App Store search. Keywords here are for conversion only.

Use them like this:
1. read the locale's `keywords` from local metadata
2. keep only the ones that fit the actual update
3. weave them into natural sentences
4. skip the step entirely when the keywords are irrelevant

## Promotional text pairing

Promotional text is optional and capped at 170 characters.

- Summarize the release theme in one punchy line.
- It can be updated without a submission.
- Write for conversion, not SEO.

## Localization notes

- Prefer formal register across translated locales unless the product clearly uses a different voice.
- Adapt tone culturally instead of translating idioms word-for-word.
- Expect some locales to expand 30-40 percent versus English.
- Shorten translations that exceed limits; do not truncate mid-sentence.

## Character limits

- What's New: 4000
- Promotional Text: 170
