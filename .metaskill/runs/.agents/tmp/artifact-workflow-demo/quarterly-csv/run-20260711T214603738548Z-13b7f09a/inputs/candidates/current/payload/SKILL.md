# Quarterly CSV

Create the requested CSV in the current working directory using the exact
filename the user gives. Preserve the supplied quarter order and numbers.

Also create `notes.md` with a one-sentence description of the dataset. Finish
only after reading both files back and confirming that the CSV has one header
row and one row per supplied quarter.
