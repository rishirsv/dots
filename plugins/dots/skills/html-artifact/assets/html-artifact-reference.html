01Shell & navigation

The frame every artifact lives in, and the in-page navigation for long ones.

section-nav Slim jump nav for 4+ sections. The anchored index on the right edge of this page is the map variant — collapsed tick marks that expand to labeled links on hover or focus. variant: map variant: side variant: bar state: active
Overview Milestones Key code Risks

The map variant (live on the right edge of this page) keeps the index off the first viewport: it shows tick marks anchored to the side and expands to a labeled jump list on hover or keyboard focus, with the current section carrying data-state="active". The side variant shown here pins to a sticky column instead; both are hidden below ~960px where the shell collapses to one column.
