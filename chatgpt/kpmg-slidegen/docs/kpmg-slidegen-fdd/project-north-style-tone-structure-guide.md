## Tone Rules
- Executive diligence voice: neutral, precise, and implication-led.
- Avoid promotional phrasing; assertions should be auditable against data or source materials.
- Frequent term families: adjustment, reported, transaction, margin, pro forma, working capital, run-rate, reconciliation, support, bridge, variance, risk, assumption, normalized

## Structure Rules
- Chart/table-left layouts: left side is exhibit placeholder; right side carries interpretation and implications.
- Two-column bullet layouts: left side sets context/definitions, right side carries findings and decision relevance.
- One-column analytical pages: sequence as observation -> evidence -> implication -> risk/sensitivity note.

## Syntax Rules
- Use complete sentences with explicit punctuation for manager-review readability.
- Prefer two-sentence analytical bullets: sentence one states movement/evidence, sentence two states implication.
- Include period references (FY/TTM/quarter) whenever trends are discussed.
- Use colons for structured labels (for example `Revenue:` or `Adjustment 1:`), but avoid over-segmentation.

## Numeric and Evidence Rules
- Preserve key facts from source inputs verbatim when fact lock applies.
- Include scale qualifiers and units (`$`, `%`, `m`, `'000`) when relevant to interpretation.
- For placeholder-based slides, increase right-side narrative density to compensate for deferred exhibits.

## Blocked Patterns
- Sparse, generic bullets that do not explain why the metric matters.
- Number drift, entity drift, or transformed values that break source fidelity.
- Placeholder mutation (editing required tokens such as chart/table markers).

## Review Rubric
1. Layout fit: does the selected layout match intent and section role?
2. Evidence quality: are key claims tied to numbers, periods, and support language?
3. Syntax quality: are bullets sentence-complete and client-ready?
4. Implication quality: does each major point explain transaction relevance?
5. Fidelity: are source facts preserved with no numeric/entity drift?
