# Slide skill pattern review plan

## Phases and outcomes
1. Asset discovery  
   - Outcome: Confirm which slide/presentation skills already exist so we know where to mine golden patterns.
2. Pattern capture  
   - Outcome: Collect reusable workflow, QA, and formatting language verbatim so the KPMG slide skill can mirror those rhythms.
3. Recommendations  
   - Outcome: Explain how the captured patterns can inform outline-mode structure, QA/overflow checks, output examples, and writing guidance for the KPMG skill.
