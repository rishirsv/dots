# KPMG Slides doc review plan

## Phase outcomes
1. Review the existing skill docs so we know what guidance is already provided and how it flows.
2. Highlight gaps vs effective slide creation practices and capture precise problem statements.
3. Turn each gap into actionable edit notes so the docs can be revised without a heavy rewrite.

## Implementation checklist
- [ ] 1.0 Document audit
  - [ ] 1.1 Read SKILL.md and references to capture scopes and inventory
  - [ ] 1.2 Note missing best-practice coverage (outline, schema checks, QA depth, tool affordances)
- [ ] 2.0 Gap reporting
  - [ ] 2.1 Quote problematic passages for context
  - [ ] 2.2 Organize gaps by file for traceability
- [ ] 3.0 Edit recommendations
  - [ ] 3.1 For each affected file, propose new section headings or bullets
  - [ ] 3.2 Ensure edits mention outline mode, QA rigor, overflow, schema coverage
