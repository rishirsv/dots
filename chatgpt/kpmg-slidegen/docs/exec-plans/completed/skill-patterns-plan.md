# Skill pattern extraction plan

## Phases and outcomes
1. Asset discovery
   - Outcome: Confirm which SKILL definitions are available so we know the scope for pattern mining.
2. Pattern extraction
   - Outcome: Capture the structural and QA-oriented language that makes those SKILLs reusable for slide generation.
3. Recommendations
   - Outcome: Frame how to adapt those patterns for the KPMG slide-generation skill, focusing on outline mode, QA loops, overflow guidance, and schema clarity.

- [ ] 1.0 Asset discovery

  - [ ] 1.3 Confirm no additional instructions prevent scanning
- [ ] 2.0 Pattern extraction
  - [ ] 2.1 Pull verbatim snippets (<=5 lines each) that show structure, checklists, QA, tool guidance

- [ ] 3.0 Recommendations

  - [ ] 3.3 Double-check instructions for doc, QA, and skill use
