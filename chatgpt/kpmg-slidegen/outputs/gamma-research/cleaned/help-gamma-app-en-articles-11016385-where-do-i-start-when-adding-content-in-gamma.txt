This guide walks you through all the available options. 🌀 Insight: You can type directly into a card, use AI suggestions, or access formatting tools for enhanced content creation. How to Add Content in GammaOption 1: Type Directly into a CardClick anywhere inside a card.Start typing to add text.Format your text using the toolbar that appears.   Option 2: Use Agent to Generate and Edit ContentClick the sparkle icon on a selected card, or click Agent at the top of the presentationChoose whether to refine existing content or generate new content.If generating new content, enter a prompt or select AI-generated suggestions.Apply the AI-generated text to your gamma. Option 3: Use the Slash Command to Add ContentClick inside a card and type /.A menu of available content options will appear (e.g., GIFs, images, code blocks, team photos).Select an option to insert it directly into the card. Option 4: Drag Content from the Insert MenuOpen the Insert Menu on the right-hand side of the screen.Drag and drop images, videos, embeds, or other media into your gamma.Adjust the content placement as needed.  FAQs & Common IssuesWhy can’t I type inside a card?Ensure the card is selected before typing.If text formatting options don’t appear, try refreshing the page.How do I undo an AI-generated change?Use Command + Z (Mac) or Ctrl + Z (Windows) to revert to the previous version.What if the slash menu doesn’t appear?Make sure you're typing / inside an editable text area.Try clearing your browser cache and reloading the gamma.Can I move inserted media after adding it?Yes!
