# Working capital template

## Purpose
Assess normalized working capital and implications for cash flow, peg, and close mechanics.

## Recommended structure

### Working capital definition
Start with a definition table:
- Included accounts
- Excluded accounts
- Rationale for exclusions

### Historical working capital behavior
Depending on data availability:
- Monthly/quarterly working capital trend (best for seasonality)
- Or period-end snapshots with caveats

### Normalized vs actual
- Provide a normalized estimate/range if possible, with basis.
- If not possible, explicitly state what data is missing and why it matters.

### Implications
- Potential peg sensitivity (qualitative or quantitative)
- Close-date considerations (seasonality, unusual spikes)

### Open items
- Missing AR/AP aging, inventory detail, deferred revenue detail, etc.

## Suggested exhibits
- Working capital build and normalization table
- Seasonality chart (if data exists)

## Common pitfalls
- Presenting a single point estimate without acknowledging seasonality.
- Using an unclear definition of working capital.
