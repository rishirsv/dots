Seed plan extracted from the later-roadmap rows in
`.agents/artifacts/meta-skill-evaluation-report.html`. Treat this file as the
pre-Ultraplan baseline; the upgraded plan is
`.plans/meta-skill-prs-8-14-15.ultra.md`.

8. Blind comparison: add randomized pairwise comparison grader mode so a
   reviewer or judge can compare candidate outputs without knowing which
   candidate produced which output.
14. Review report: add an HTML review report with side-by-side outputs, grades,
   trace pointers, and human-review status so a reviewer can decide from one
   page.
15. Optimizer lane: add a manual or semi-automatic bounded candidate-edit loop
   with held-out validation gates so edits are accepted only when validation
   improves and gates hold.
